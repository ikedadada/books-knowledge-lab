# 相反する願望と空集合

複数の価値観や願望が互いに制約を課し、すべてを同時に満たす現実的な行動が存在しなくなる状態を、本書は「空集合」と呼ぶ。

この状態では、外部環境ではなく内部の対立そのものが最重要ポイントになる。意思決定者は思考停止するか、相互に矛盾する中途半端な案を併存させやすい。

解を作るには、少なくとも一つの願望または制約を緩和・放棄しなければならない。全員を満足させる表現で対立を覆い隠しても、実行段階で矛盾が再発する。
