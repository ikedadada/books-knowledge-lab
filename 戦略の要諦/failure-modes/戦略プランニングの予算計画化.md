# 戦略プランニングの予算計画化

戦略プランニングと称しながら、実際には売上・利益の予測、業績目標、予算配分だけを作る失敗である。

財務計画は必要だが、重大な課題の原因、競争環境の変化、解決を阻む制約を扱わなければ戦略にはならない。ビジョンやミッションの文言作成も、課題解決の方針を生まない限り同様である。

長期計画が有効なのは、大きな流れをある程度予測でき、将来のために現在の資源を投じる意思がある場合に限られる。予測不能な状況では、シナリオを美しい一般論で終わらせず、具体的な制約や不都合な事態まで検討する必要がある。
