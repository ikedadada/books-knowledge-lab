# 短期財務指標への過集中

四半期利益や株価予想へ経営陣の注意を集中し、将来価値を作る戦略課題から目を逸らす失敗である。

現在の利益は主に過去の投資、判断、競争上の地位から生じる。一方、現在の研究開発、採用、能力構築は将来の利益に影響する。この時間差を無視すると、短期予想を満たすために将来の競争力を削ることになる。

また株価は将来利益の不確実な推定であり、多くのノイズを含む。経営行動と短期株価の間には確実な因果関係がないため、株主価値という結果だけを目標にしても行動方針は定まらない。
