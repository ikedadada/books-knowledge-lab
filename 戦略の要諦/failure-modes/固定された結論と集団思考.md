# 固定された結論と集団思考

望ましい結果や採用したい行動が事前に決まっていると、戦略会議は診断の場ではなく、既定の結論を正当化する場になる。

この状態では、反証情報が軽視され、選択肢が狭められ、早すぎる合意が形成される。楽観バイアス、確証バイアス、自分の経験を過大評価する経験バイアスがこれを強める。

本書は、問題を単なる集団思考だけに帰さない。より根本的な原因は、戦略を「決まった目標を実現する手段の選択」と誤認している点にある。複雑な課題では、目標、問題設定、選択肢そのものを再考できなければならない。
