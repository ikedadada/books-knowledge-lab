# 競争優位を生む非対称性

競争上の強みは、競争相手との非対称性から生じる。単なる得意分野ではなく、利益や課題攻略への貢献を合理的に期待でき、相手が容易に模倣できない差であることが必要になる。

本書が挙げる基本的な探索領域は五つある。

1. **情報**：他社が知らない事実や顧客理解。
2. **ノウハウ**：固有のスキル、技術、特許。
3. **地位**：ブランド、評判、市場や流通上の位置。
4. **効率**：規模、経験、技術などによるコスト差。
5. **組織運営**：機敏さ、連携、意思決定能力。

この五つは強みの存在を証明する分類ではなく、非対称性を探す領域である。利益を生むことを合理的に期待でき、最重要ポイントの攻略に利用できるかを判断する必要がある。

本書はこれらの基本要素に加えて、既存の知識と新しい需要を「結びつける」、一体だったシステムを「切り離す」といった優位性の作り方も論じる。

非対称性は、[[mechanisms/競争優位の作用原理]]に挙げる仕組みを通じて持続的な優位へ変わる。
